Figma AI Prompt

Create a three-panel desktop layout with clear visual hierarchy.

Left Panel – File Management

 File list panel showing loaded images
 Drag-and-drop support for folders and individual images
 File sorting controls (name, date, type)
 File filtering options (PNG, JPG, WEBP)
 Adjustable thumbnail size control

Center Panel – Image Preview

 Thumbnail preview grid of all images in the batch
 Live preview updates when background removal completes
 Large preview area for the currently selected image
 Ability to draw a protect / keep region rectangle on the preview image

Right Panel – Processing Settings
Organize settings into collapsible sections:

Removal Settings

 Removal mode selector
 Standard AI background removal
 Color-based background removal
 Color picker / eyedropper
 Threshold / tolerance slider
 Dynamic UI (color controls only visible when color mode is selected)

Subject Protection

 Protect / keep region tool
 Option for per-image protect region

Manual Editing Tools

 Erase brush
 Restore brush
 Edge refinement brush

Edge & Quality Controls

 Feather / softness slider
 Smooth edges toggle
 Decontaminate colors toggle
 Alpha matting controls

AI Model Selection

 Model selector dropdown with:

   u2net
   u2netp
   isnet-general-use
   SAM segmentation
 Speed vs quality toggle

Background Output Options

 Transparent background (PNG)
 Solid color replacement with color picker
 Image background replacement

Bottom Status Bar

 Progress bar for batch processing
 Cancel processing button
 Status / log panel showing processing steps

Export & Output Controls

 Output directory selector
 Output format selector (PNG, JPG, WEBP)
 Naming scheme selector
 Optional export results as ZIP

Automation Features

 Watch folder mode toggle
 Auto-process new images option

Performance Options

 Resize large images before processing
 Sequential vs parallel processing toggle
 Memory-safe mode toggle

Visual Style

 Clean modern desktop application design
 Inspired by professional creative software (Adobe / Figma style layout)
 Dark mode and light mode variants
 Rounded panels and subtle shadows
 Clear icons for tools (brush, color picker, AI model, export)
 Minimal clutter, strong spacing, intuitive workflow
